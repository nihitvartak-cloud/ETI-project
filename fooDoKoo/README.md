# fooDoKoo

A simple UI website for swapping non-veg recipes to veg with a pseudo-AI suggestion bar.

## Features

- Paste a non-veg recipe and convert it to a vegetarian version using basic substitutions.
- Suggest new and extraordinary vegetarian dishes based on a keyword.

## How to run

This is a static site. Open `index.html` in your browser to use the app. No server required.

## Future Enhancements

- Replace placeholder conversion and suggestion logic with real AI/back-end calls.
- Add persistent storage for user recipes and suggestions.
- Improve UI/UX and accessibility.
